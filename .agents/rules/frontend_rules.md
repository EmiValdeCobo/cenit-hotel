# Reglas de Workspace para Frontend

Siempre que se trabaje bajo el rol del agente `@frontend` o se realicen modificaciones en archivos dentro de la carpeta `frontend/`, se debe cumplir obligatoriamente lo siguiente:

- **Habilidad Requerida**: Leer y aplicar la guía en [frontend_nextjs/SKILL.md](file:///c:/Users/emili/OneDrive/Desktop/cenit/skills/frontend_nextjs/SKILL.md).
- **Control de Componentes**: Respetar estrictamente la separación de React Server Components y Client Components.
- **Estilo**: Aplicar las pautas de Tailwind CSS para diseño responsivo e interactividad premium.
